## SETUP

### Clone project

### Install Application Dependencies
```bash
npm install
```

### Run client
```bash
npm run client
```

### Run migration for server
```bash
npx sequelize-cli db:migrate
```

### Run seeding for server
```bash
npx sequelize-cli db:seed:all
```

### Run server
```bash
npm run server
```

### Run server and client concurrently
```bash
npm run dev
```
